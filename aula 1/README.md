# Aula 1 e 2: Introdução a Algoritmos e Meu Primeiro Algoritmo

Nestas aulas, aprendi a definição de um algoritmo, como ele está presente em nossas vidas e a importância de seu uso correto. Estudamos as formas de representação através de fluxogramas e pseudocódigos, compreendendo a diferença entre eles e as linguagens de programação. Além disso, avancei para os conceitos iniciais de manipulação de dados com o uso de variáveis e comandos de saída.

---

## 📚 O que foi aprendido?

### Conceitos Básicos
* **Definição de Algoritmo:** Entendimento de que um algoritmo é uma sequência finita de passos lógicos e bem definidos para resolver um problema ou executar uma tarefa.
* **Algoritmos no Cotidiano:** Identificação de como os algoritmos estão presentes no nosso dia a dia (como em uma receita de bolo ou no trajeto para o trabalho).
* **Uso Correto:** A importância da precisão, clareza e ordem na construção de um passo a passo para evitar ambiguidades.

### Ferramentas de Representação
* **Fluxograma:** Representação gráfica e visual de um algoritmo utilizando formas geométricas padrões (início, decisão, processos, etc.).
* **Pseudocódigo:** Uma forma de representar o algoritmo usando uma linguagem intermediária (próxima à nossa língua nativa), sem as regras rígidas de uma linguagem de programação específica.
* **Diferença para Linguagem de Programação:** Compreensão de que o pseudocódigo serve para planejar a lógica, enquanto a linguagem de programação é o código real que o computador consegue interpretar e executar (como Python, JavaScript, C, etc.).

### 💻 Introdução à Programação e Sintaxe (Portugol)
* **Conceito de Variável:** Entendimento de que uma variável é um espaço alocado na memória do computador para armazenar dados temporariamente que podem ser modificados durante a execução do programa.
* **Tipos Primitivos de Dados:** Aprendizado sobre os diferentes tipos de dados que uma variável pode armazenar (como `Inteiro`, `Real`, `Caractere` e `Logico`).
* **Regras de Nomeação (Identificadores):** Compreensão do que uma variável pode ou não receber como nome (regras de início com letras/underline, proibição de espaços, caracteres especiais, acentos e palavras reservadas do sistema).
* **Comandos de Saída:**
  * `escreva`: Comando utilizado para exibir textos ou valores na tela, mantendo o cursor na mesma linha.
  * `escreval`: Comando que também exibe dados na tela, mas adiciona uma quebra de linha automática ao final (escreva + linha).
  * **Manipulação de Textos:** Fixação da regra de que tudo o que estiver escrito dentro dos parênteses e entre aspas nos comandos de saída é interpretado pelo programa e exibido literalmente como texto na tela.

### 🔍 Pesquisa e Aprofundamento Pessoal
Durante as aulas, busquei aprofundar o entendimento diferenciando três pilares essenciais:
* **Raciocínio Lógico:** A habilidade de pensar de forma estruturada e coerente para encontrar a solução de um problema.
* **Solução Estruturada:** A organização e o desenho estratégico de como resolver o problema, antes mesmo de escrever as instruções.
* **Algoritmo:** A tradução prática dessa solução em um passo a passo sequencial e lógico.
